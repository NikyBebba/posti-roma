# Posti Roma

Guida interattiva ai posti di Roma.

🌐 **Sito live:** https://nikybebba.github.io/posti-roma/

## Sviluppo

Il sito è una singola pagina statica (`index.html`) pubblicata tramite GitHub Pages
dal branch `main`, cartella root.

Per aggiornare il sito:

```bash
# modifica index.html
git add index.html
git commit -m "Aggiorna la guida"
git push
```

GitHub Pages ricostruisce automaticamente il sito a ogni push su `main`.
